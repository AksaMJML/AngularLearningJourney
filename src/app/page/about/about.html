About Demo Angular

Demo Angular is a simple and clean Angular application created to help you understand components, routing, and UI design in a practical way.

Who We Are

We focus on building lightweight, modern, and responsive Angular layouts. This demo project showcases how to structure pages, use components, and maintain a consistent theme across your application.

What This Project Includes

  • Reusable components (Navbar, Footer, Cards)
  • Clean and modern UI design
  • Simple routing setup
  • Light gray themed pages for better readability

Our Mission

To help beginners learn Angular through clean and simple examples.

Our Vision

To create easy‑to‑understand UI layouts that anyone can build on.

Our Values

Clarity, simplicity, and modern design principles.
